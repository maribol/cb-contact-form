== Description ==

A very simple contact form plugin for Wordpress

Just install the plugin and add it as a widget to wherever you want the form to appear.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `the directory "cb-contact-form"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance" and then to Widgets and add the CB Contact Form in any widget section.
